# S3

Creates an S3 bucket with server-side encryption, optional versioning, CORS, and CloudFront access. Bucket names are auto-suffixed with the AWS account ID for global uniqueness.

## Usage

```hcl
module "media" {
  source = "../../modules/s3"

  name_prefix = "myapp-staging"
  bucket_name = "media"

  # Optional
  cloudfront_access    = module.cloudfront.distribution_arn
  cors_allowed_origins = ["https://myapp.example.com"]
}
```

## Key Variables

| Variable             | Type         | Description                                          |
| -------------------- | ------------ | ---------------------------------------------------- |
| name_prefix          | string       | Prefix for resource names                            |
| bucket_name          | string       | Bucket name (combined: {prefix}-{name}-{account_id}) |
| versioning           | bool         | Enable versioning (default: false)                   |
| public               | bool         | Allow public access (default: false)                 |
| cloudfront_access    | string       | CloudFront distribution ARN for bucket policy        |
| cors_allowed_origins | list(string) | CORS allowed origins                                 |

## Outputs

| Output                      | Description                                   |
| --------------------------- | --------------------------------------------- |
| bucket_name                 | Full bucket name                              |
| bucket_arn                  | Bucket ARN                                    |
| bucket_domain_name          | Bucket domain name                            |
| bucket_regional_domain_name | Regional domain name (for CloudFront origins) |
